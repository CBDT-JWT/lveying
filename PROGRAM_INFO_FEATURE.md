# 节目详情功能说明

## 🎭 新功能概述

节目单现在支持为每个节目添加详细的介绍信息，使用 Markdown 格式，支持**粗体**和*斜体*文本样式。

## ✨ 功能特性

### 1. Markdown 格式支持
- ✅ 粗体：使用 `**文本**` 或 `__文本__`
- ✅ 斜体：使用 `*文本*` 或 `_文本_`
- ✅ 换行：直接按回车即可
- ✅ 实时预览：编辑时可以看到渲染效果

### 2. Guest端展示
- ✅ 点击节目卡片展开查看详情
- ✅ 优雅的展开/收起动画
- ✅ 美观的详情展示面板
- ✅ 支持移动端触摸操作

### 3. Admin端管理
- ✅ 为任意节目添加/编辑详情
- ✅ 实时预览 Markdown 渲染效果
- ✅ 编辑时不影响节目状态
- ✅ 一键保存到数据文件

## 📋 使用指南

### Admin端操作

#### 1. 进入节目管理
访问：`http://localhost:3000/admin/programs`

#### 2. 编辑节目详情
1. 找到要编辑的节目
2. 点击 **"📝 编辑详情"** 按钮
3. 在文本框中输入 Markdown 格式的内容

#### 3. Markdown 格式示例
```markdown
这是一个**精彩**的节目

表演时间：*20分钟*
表演人数：*8人*

**节目亮点**：
- 融合了传统与现代元素
- *获得过多项大奖*
- **不容错过**的精彩表演
```

#### 4. 预览和保存
- 输入内容后会实时显示预览效果
- 确认无误后点击 **"💾 保存"** 按钮
- 保存成功会显示提示信息

### Guest端查看

#### 1. 访问节目进度页面
访问：`http://localhost:3000/programs`

#### 2. 查看节目详情
1. 找到有 **"▼ 点击查看详情"** 提示的节目
2. 点击节目卡片展开详情
3. 再次点击可以收起详情

## 💡 Markdown 语法快速参考

### 粗体
```markdown
**这是粗体文本**
__这也是粗体文本__
```
效果：**这是粗体文本**

### 斜体
```markdown
*这是斜体文本*
_这也是斜体文本_
```
效果：*这是斜体文本*

### 粗体+斜体
```markdown
***这是粗体加斜体***
**_混合使用_**
```
效果：***这是粗体加斜体***

### 换行
直接按回车键即可换行，会自动显示为 `<br />`

## ⚠️ 注意事项

### 1. Markdown 语法
- 粗体和斜体符号必须成对出现
- 符号和文本之间不能有空格
- ❌ 错误：`** 文本 **`
- ✅ 正确：`**文本**`

### 2. 内容建议
- 保持简洁，避免过长的段落
- 突出重点信息（时长、人数、特色等）
- 使用粗体强调关键词
- 使用斜体标注细节信息

### 3. 移动端显示
- 内容会自动适配移动屏幕
- 建议单行不要超过 30 个汉字
- 适当使用换行提高可读性
